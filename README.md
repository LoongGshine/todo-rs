## todo-rs

A minimalist CLI todo list write by rust.

## Usage

```
Usage: rodo [command] [options]

Commands:
  add [contents]        Add a todo item
  rm [item_id]          Remove a todo item
  ls                    List all the todo items
```

## Development

```
# clone the repo
git clone

# run the program in local
cargo run

# build the program
cargo build
```
